# Django Local Chatbot

โปรเจกต์นี้ใช้ Chainlit เป็นหน้าแชตสำหรับระบบ Django + Ollama + Chroma RAG ชุดเดิม

สิ่งที่ทำได้จากหน้านี้:

- คุยกับ local model ได้โดยตรง
- ใช้ฐานความรู้ที่เพิ่มไว้ในระบบเดิมอัตโนมัติ
- บัญชี `user` ใช้ถามคำถามจากฐานความรู้แบบ `shared` ได้อย่างเดียว
- บัญชี `admin` อัปโหลด แก้ไข และลบเอกสาร `shared` ได้จากหน้าแชต
- ระบบใช้ข้อมูลจาก RAG เป็นหลักก่อน ถ้าไม่พบข้อมูลในฐานความรู้จึงค่อย fallback ไปใช้ความรู้ทั่วไปของโมเดล
- คำตอบจะยึดจากข้อมูลใน RAG เป็นหลัก และจะใช้ประวัติคำถามก่อนหน้าเฉพาะเมื่อคำถามปัจจุบันเป็น follow-up หรือกำกวม เพื่อช่วยตีความเท่านั้น
- ถ้าคำถามใหม่ขึ้นต้นแบบ follow-up เช่น `แล้ว...` หรือ `กรณีนี้...` ระบบจะนำคำถามก่อนหน้าของผู้ใช้มาช่วยค้น RAG และจัดลำดับผลที่เกี่ยวข้องขึ้นก่อน โดยไม่ตัดผลอื่นทิ้งทั้งหมด
- ล็อกอินด้วยบัญชี Django ได้
- บันทึกประวัติข้อความลงฐานข้อมูลแยกตามผู้ใช้และตามห้องสนทนา
- ดูรายการห้องสนทนาของตัวเองได้จาก sidebar ด้านซ้าย
- เปิดห้องเก่ากลับมาคุยต่อได้จาก sidebar
- เริ่มห้องสนทนาใหม่ได้จากปุ่ม New Chat ของ Chainlit
- ลบห้องสนทนาของตัวเองได้จาก sidebar
- เปลี่ยนชื่อห้องสนทนาของตัวเองได้จาก sidebar
- คำตอบจาก AI แสดงแบบ streaming ทีละส่วน ทำให้รออ่านได้ทันที
- หลังล็อกอินจะมี badge มุมบนขวาแสดง `Role: Admin` หรือ `Role: User`
- แก้ไขคำถามล่าสุดของตัวเองได้ และระบบจะ regenerate คำตอบใหม่จากจุดนั้น

ถ้ายังไม่มีบัญชี ให้สร้างผู้ใช้จาก Django ก่อน เช่น `python manage.py createsuperuser`

หน้า Login ของ Chainlit ใช้ได้ทั้ง `username` และ `email` ของบัญชี Django

ถ้าพร้อมแล้ว ล็อกอินและพิมพ์คำถามภาษาไทยได้เลย โดยประวัติห้องสนทนาจะอยู่ที่ sidebar ด้านซ้ายของหน้าจอ
